### 🎮 Steuerung (Keywords für Render.com)

| Keyword | Phase | Farbe |
| :--- | :--- | :--- |
| `slowdown` | Verlangsamung | 🟠 Orange |
| `recovery` | Erholung | 🟢 Grün |
| `expansion` | Expansion | 🟡 Gelb |
| `recession` | Rezession | 🔵 Blau |
| `inflation` | Inflation | 🟢 Grün |
